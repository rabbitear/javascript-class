Project #1: Is simple pictures of china that scroll through using a JQuery plugin called Cycle. This was my first time using a plugin for JQuery.

Project #2: Is a rock scissors paper game. While running through the CodeCademy tutorials on javascript they have you build a simple rock scissors paper game. I took this game logic and expanded it to an a lot more playable game with extra bells and whistles.

Project #3: This project is probably not complete, I was going use this terminal plugin that I found for JQuery, which I think was cool and could make some kind of adventure games in text or something, but this specific project I did not have time for.

Project #4: This project actually started a while back, since I knew a little html and css already. I did not know much of any JavaScript at the time, so I recently added new code to make it alot more attractive hopefully.

Project #5: Is a minimal Animal Search, you type in a name and it comes up with a picture of that name. There are not too many animals, I'm mostly trying out how to input text for search etc.

Project #6: This is suppose to be a Youtube Search for looking up videos, you can do that on regular youtube sight, but the youtube site is always slow for me on my computer so made this search as something more light weight in the browser.

Final Project: Explained a little bit in the intro paragraph, this is the Chinese Flash cards, where I can flash up a card, you try and guess it, then you can trun it over for more information.
